# Real-Time-Chat-Room
Full-Stack Chat-Room Application using Springboot, React, Websocket and Mysql

## 🚀 Features
**Messaging System**
- 🔍 Users Create or Join Rooms By Entering Room ID
- 📝 Seamless Creation of New Chat Rooms
- 💬 Real-Time Messaging Between Users
